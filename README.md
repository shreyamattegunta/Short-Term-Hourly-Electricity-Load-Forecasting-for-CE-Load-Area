## Short-Term Hourly Electricity Load Forecasting

### Overview
This project develops short-term hourly electricity load forecasts using historical data. It applies statistical time series models such as ARIMA and Exponential Smoothing to capture trends and seasonality. The results demonstrate how accurate forecasting can support efficient, reliable, and data-driven utility operations.

### Requirements
R (latest version recommended)

Libraries:
- forecast
- tseries
- ggplot2

Install required packages:

```r
install.packages(c("forecast", "tseries", "ggplot2"))
```

### Project Highlights

- Captures strong daily, weekly, and seasonal demand patterns
- Compares multiple time series models for forecasting accuracy
- Demonstrates practical applications of forecasting in energy systems
